Swift lazy collections

Swift lazy collections 会延迟操作。这对于大集合来说，可以减少内存的使用和提高性能。但是，对于简单的集合，我们使用数组的效率会更高。

let numbers = [1, 2, 3].lazy.map { $0 * $0 }
print(numbers) // LazyMapCollection<Array<Int>, Int>
